Greater Than (>)

> 𝕩: Merge

Add the rank of an element of 𝕩 to the rank of 𝕩.

All elements must have the same rank.

Returns atomic values as is.

↗️
      a ← ⟨⟨1, 2⟩, ⟨3, 4⟩⟩

      >a
┌─     
╵ 1 2  
  3 4  
      ┘

      ≢a
⟨ 2 ⟩

      ≢>a
⟨ 2 2 ⟩

𝕨 > 𝕩: Greater Than

𝕨 and 𝕩 can both be either numbers or characters.

Pervasive.

↗️
      1 > 3
0

      2‿3‿0 > 3‿1‿0
⟨ 0 1 0 ⟩

      'a' > 'b'
0
