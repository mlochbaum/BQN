Exclamation Mark (!)

! 𝕩: Assert

→full documentation

Throw an error if 𝕩 is not 1.

↗️
    ! 1
1

    ! 2
Error: Assertion error

    ! "hello"
Error: hello

𝕨 ! 𝕩: Assert With Message

→full documentation

Throw an error with message 𝕨 if 𝕩 is not 1.

↗️
    "hi" ! 1
1

    "two" ! 2
Error: two

    "hello error" ! "hello"
Error: hello error
