Left Curly Bracket ({)

{ ...: Begin Block

→full documentation

Starts a block, which can be one of:

  • Function
  • 1-Modifier
  • 2-Modifier
  • Namespace
  • Immediate Block

Must end with a corresponding }.

↗️
    {𝕨 + 𝕩}   # Function
(function block)

    {𝕨‿𝔽‿𝕩}   # 1-modifier
(1-modifier block)

    {𝕨‿𝔽‿𝔾‿𝕩} # 2-modifier
(2-modifier block)

    {a ⇐ 5}   # Namespace
{a⇐}

    {5+4+6}   # Immediate block
