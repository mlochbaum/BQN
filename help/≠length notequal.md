Not Equal (`≠`)

`≠ 𝕩`: Length

Length of the first dimension of `𝕩`.

```
  ≠ 3
1
  ≠ ⟨1, 2, 3⟩
3 
  ≠ 3‿4‿5⥊0
3 
  ≠ 1‿4‿5⥊0
1 
  ≠ 4‿4‿5⥊0
4
```

```
  1 ≠ 3
1
  2‿3‿0 ≠ 3‿1‿0
⟨ 1 1 0 ⟩
  'a' ≠ 'a'
0
```
