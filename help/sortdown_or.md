# Logical Or (`∨`)
    
## `∨ 𝕩`: Sort Down
    
Sort array `𝕩` in descending order.
    
          ∨ 3‿1‿4‿1‿5

    
    
## `𝕨 ∨ 𝕩`: Logical Or
    
Logical Or of `𝕨` and `𝕩`. 
    
[Pervasive.](../doc/arithmetic.md#pervasion)
    
          1 ∨ 0

          1‿0 ∨ 1‿1

          0 ∨ 0
