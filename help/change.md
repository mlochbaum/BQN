# Leftwards Arrow With Hook (`←`)

## `n ↩ v`: Change

Changes the value of variable with name `n` to value `v`.

Variable `n` must already exist.

        a ↩ 1

        ⊢ b ← 3

        ⊢ b ↩ "Be the change you wish to see in the world."
