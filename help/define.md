# Leftwards Arrow (`←`)

## `n ← v`: Define
[→full documentation](../doc/expression.md#assignment)

Defines a new variable with name `n` and value `v`.

Variable `n` must not already exist in the scope.

        ⊢ a ← 1

        ⊢ b ← 3‿3 ⥊ 5

        C ← ↑
