# Square Image Of Or Equal To (`⊑`)

## `⊑ 𝕩`: First

First element of `𝕩`.

        ⊑ ⟨1, 2, 3⟩

        a ← 3‿3 ⥊ ↕9

        ⊑ a



## `𝕨 ⊑ 𝕩`: Pick

Pick the element of `𝕨` at index `𝕩`.

        2 ⊑ ⟨1, 2, 3⟩

        b ← 3‿3 ⥊ ↕9

        2‿0 ⊑ b
