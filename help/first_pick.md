# Square Image Of Or Equal To (`⊑`)

## `⊑ 𝕩`: First
[→full documentation](../doc/pick.md#first)

First element of `𝕩`.

        ⊑ ⟨1, 2, 3⟩

        a ← 3‿3 ⥊ ↕9

        ⊑ a



## `𝕨 ⊑ 𝕩`: Pick
[→full documentation](../doc/pick.md)

Pick the element of `𝕩` at index `𝕨`.

        2 ⊑ ⟨1, 2, 3⟩

        b ← 3‿3 ⥊ ↕9

        2‿0 ⊑ b
