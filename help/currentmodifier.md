# Mathematical Double-struck R (`𝕣`)

## `𝕣`: Current Modifier

A variable assigned to the current modifier block. Underscores must be added to the beginning and/or end (`_𝕣`, `_𝕣_`) to use the modifier with the correct role.


        +{𝕣⊣𝕩} 4
