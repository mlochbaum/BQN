# Equal (`=`)

## `= 𝕩`: Rank

Returns the number of dimensions in `𝕩`.


        = 0

        = 3⥊0

        = 3‿3⥊0

        3‿3‿3 ⥊ ⟨⟨0⟩⟩



## `𝕨 = 𝕩`: Equal To

[Pervasive.](../doc/arithmetic.md#pervasion)

        1 = 3

        2‿3‿0 = 3‿1‿0

        'a' = 'a'
