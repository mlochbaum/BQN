# Right Curly Bracket (`}`)

## `... }`: End Block
[→full documentation](../doc/block.md)

Starts a block, which starts with `}`. See [Begin Block](beginblock.md) for more details.

        {𝕨 + 𝕩}   # Function

        {𝕨‿𝔽‿𝕩}   # 1-modifier

        {𝕨‿𝔽‿𝔾‿𝕩} # 2-modifier

        {a ⇐ 5}   # Namespace

        {5+4+6}   # Immediate block
