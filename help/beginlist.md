# Mathematical Left Angle Bracket (`⟨`)

## `⟨ ...`: Begin list
[→full documentation](../doc/arrayrepr.md#brackets)

Starts a list. Inner elements must be separated by `,` or `⋄`.
Lists can be nested in other lists.

Must end with a corresponding `⟩`.

        ⟨1, 2, 3⟩

        ⟨+ ⋄ - ⋄ 56⟩
