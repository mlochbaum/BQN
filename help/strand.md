# Undertie (`‿`)

## `‿`: Strand
[→full documentation](../doc/arrayrepr.md#strands)

Create a list via strand notation.

Placing `‿` between valid BQN expressions will create a list out of the end results of those expressions.


        1‿2‿3

        +‿-‿56
